Voltar

1 — Abrir o Terminal Integrado

No VS Code, pressione Ctrl + ` (crase) ou vá em Exibir → Terminal.

2 — Acessar a pasta do repositório

No terminal, entre na pasta onde está o seu projeto:

cd caminho/para/o/repositorio

3 — Inicializar repositório (se ainda não existir)

Se a pasta não tiver o controle do Git, inicialize:

git init

Depois, conecte ao repositório remoto (GitHub):

git remote add origin git@github.com:SEU_USUARIO/nome-do-repositorio.git

4 — Abrir arquivos para editar

Abra um arquivo específico:

code index.html

Ou abra toda a pasta do projeto:

code .

5 — Verificar status e enviar alterações

Veja o que foi modificado:

git status

Adicione e envie as alterações para o GitHub:

git add .
git commit -m "Atualiza arquivos"
git push origin main

6 — Atualizar projeto em outros computadores

Antes de editar, sempre atualize seu repositório local:

git pull origin main
